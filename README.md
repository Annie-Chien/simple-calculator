# Simple Calculator

[live demo](https://vanilla-js-simple-calculator.netlify.app)

## Features
1. Basic operations (addition, subtraction, multiplication and division)
2. Keyboard support

## Usage Instruction for Keyboard
- addition ( + )
- subtraction ( - )
- multiplication ( * )
- division ( / )
- equal ( = or Enter )
- decimal point ( . )
- clear ( C or c )
